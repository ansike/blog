### docker基础命令
```shell
# 镜像相关
docker images
# docker image rm [imageId]

# 容器相关
docker ps -a
# docker container rm [containerId]
# docker {start|stop} [containerId]

# 运行
docker run 

# 编译
docker build
# docker build -f [dockerfile path] -t [name:tag] .(build上下文位置)

# 进入容器查看
docker exec -it [containerId] bash
```
