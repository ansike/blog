##### 测试语法
1. -d 判断是目录
1. -f 判断是文件
1. -e 判断目录或者文件存在
1. -r 判断文件可读
1. -s 判断文件大小不为0为真
1. -x 判断文件可执行
1. -w 判断文件可写
1. -L 文件存在且链接文件为真
1. f1 -nt f2  f1比f2新则为真
1. f1 -ot f2  f1比f2旧则为真

- 1和2是等价的
- [[]]为test拓展的命令，常用语通配符进行模式匹配。&&,||,>,<可以使用
- (())常用语计算

##### 
``` shell
[ -d public ] && echo 1
[ -s package.json ] && echo 1

#  不存在的变量必须加引号包裹
[ -f $a ] && echo 1 || echo 0 # 1
[ -f "$a" ] && echo 1 || echo 0 # 0
```
