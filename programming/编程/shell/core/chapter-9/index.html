进程管理命令

1. ps 查看进程
2. pstree 显示进程状态树
3. kill 终止进程
4. killall，pkill 通过进程名终止进程
5. top 实时显示系统中各个进程的资源占用状况
6. nohup 用户退出系统进程继续工作

1. ps 查看进程

查看某个时刻的进程快照

语法: ps [option]

参数 说明 参数 说明
-a 显示所有终端下执行的基础 u 显示进程的用户信息
a 显示与终端相关的所有进出，包括进程的完整路径 -u 显示指定用户相关的进程信息
x 显示与终端无关的所有进出 f 显示进程树
-e 显示所有进程 -f 额外显示 UID，PID，C，STIME
-l 以详细的格式显示进程的状况

PID:进程标识号
TTY:进程所属终端控制台
TIME:进程所使用总的 CPU 时间
CMD:正在执行的命令行

2. pstree 显示进程状态树

语法: pstree [option] [/]

3. kill 终止进程

语法: kill [option] [PID]

参数 说明
-l 列出全部的信号名称
-p 指定 kill 命令值打印相关进程的进程号，而不发送任何信号
-s 指定要发送的信号

常用的信号

参数 说明
HUB(1) 挂起，终端掉线或用户退出而引发
INT(2) 中断，通常是 ctrl+c
QUIT(3) 退出，通常是 ctrl+\
KILL(9) 立即结束进程的运行
TERM(15) 终止，通常在系统关机时触发
TSTP(20) 暂停进程的运行，通常是 ctrl+z

4. killall，pkill 通过进程名终止进程

语法: killall [option] [name]

参数 说明
-u 终止指定用户的进程
-w 等待被终止的进程死去

语法: pkill [option] [name]

参数 说明
-u 终止指定用户的进程
-t 杀死指定中断的进程

5. top 实时显示系统中各个进程的资源占用状况

语法: top [option]

6. nohup 用户退出系统进程继续工作

语法: nohup [option]
