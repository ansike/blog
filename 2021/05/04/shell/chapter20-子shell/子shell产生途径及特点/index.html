子shell产生途径及特点

产生的途径

  1. 带”&”提交后台作业【异步模式】。能使用本地变量
  2. 使用管道功能。能使用本地变量
  3. 使用”()”功能。能使用本地变量
  4. 通过调用外部的shell脚本产生子shell，子shell中仅能使用父shell中的环境变量，不能使用本地变量

测试shell

  1. 使用&后台执行

  1. 使用管道

  2. 使用()功能

  3. 调用外部shell

subshell

总结

  1. 父shell不能使用子shell中的变量，子shell可以使用父shell中的变量。注意区分局部变量和环境变量
  2. 管道,(),{}&都会生成子shell，注意变量使用避坑。
