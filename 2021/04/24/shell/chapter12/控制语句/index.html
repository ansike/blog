shell中的控制语句

  1. break n 跳出循环n层，省略的话跳出整个循环
  2. continue n 退到n层继续循环，省略的话忽略本次循环的剩余代码继续循环
  3. return n 用在函数里作为函数的返回值
  4. exit n 退出当前shell程序，作为上一次shell程序执行的返回值

? 好像没有发现break n和continue n于编程中的实际意义
