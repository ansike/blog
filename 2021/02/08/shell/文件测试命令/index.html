文件测试

测试语法
  1. -d 判断是目录
  2. -f 判断是文件
  3. -e 判断目录或者文件存在
  4. -r 判断文件可读
  5. -s 判断文件大小不为0为真
  6. -x 判断文件可执行
  7. -w 判断文件可写
  8. -L 文件存在且链接文件为真
  9. f1 -nt f2 f1比f2新则为真
  10. f1 -ot f2 f1比f2旧则为真
  • 1和2是等价的
  • [[]]为test拓展的命令，常用语通配符进行模式匹配。&&,||,>,<可以使用
  • (())常用语计算
