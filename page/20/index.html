shell中的控制语句

  1. break n 跳出循环n层，省略的话跳出整个循环
  2. continue n 退到n层继续循环，省略的话忽略本次循环的剩余代码继续循环
  3. return n 用在函数里作为函数的返回值
  4. exit n 退出当前shell程序，作为上一次shell程序执行的返回值

shell调用方式

调用方式分类

  1. fork方式。sh xxx.sh 同步执行所有的shell
  2. exec方式。exec xxx.sh 执行完exec脚本的内容后，父shell剩余的代码不能执行
  3. source或者.方式。source xxx.sh 同步执行所有的shell，且能读取子shell中的全部变量

子shell产生途径及特点

产生的途径

  1. 带”&”提交后台作业【异步模式】。能使用本地变量
  2. 使用管道功能。能使用本地变量
  3. 使用”()”功能。能使用本地变量
  4. 通过调用外部的shell脚本产生子shell，子shell中仅能使用父shell中的环境变量，不能使用本地变量

测试shell

  1. 使用&后台执行

  1. 使用管道

  2. 使用()功能

  3. 调用外部shell

subshell

总结

  1. 父shell不能使用子shell中的变量，子shell可以使用父shell中的变量。注意区分局部变量和环境变量
  2. 管道,(),{}&都会生成子shell，注意变量使用避坑。
