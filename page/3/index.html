常用测试语法

  1. 文件测试语法
  2. 字符串测试语法
  3. 整数二元比较操作符
  4. 逻辑操作符

文件目录测试语法

  1. -d 判断是目录
  2. -f 判断是文件
  3. -e 判断目录或者文件存在
  4. -r 判断文件可读
  5. -s 判断文件大小不为0为真
  6. -x 判断文件可执行
  7. -w 判断文件可写
  8. -L 文件存在且链接文件为真
  9. f1 -nt f2 f1比f2新则为真
  10. f1 -ot f2 f1比f2旧则为真
  • 1和2是等价的
  • [[]]为test拓展的命令，常用语通配符进行模式匹配。&&,||,>,<可以使用
  • (())常用语计算
示例示例

字符串测试语法

  1. -n “字符串” 判断字符串长度不为0为真[no zero]
  2. -z “字符串” 判断字符串长度为0为真 [zero]
  3. “字符串1” = “字符串2” 相等为真
  4. “字符串1” != “字符串2” 不等为真
  • 字符串测试时必须加引号，不加很有可能发生语法没问题，但是逻辑结果和预期的不一致
示例

整数二元比较操作符

  1. -eq 相等在test和[]中是这样，在[[]]和(())是 ==或者=
  2. -ne 不等在test和[]中是这样，在[[]]和(())是 !=
  3. -gt 大于在test和[]中是这样，在[[]]和(())是 >
  4. -lt 小于在test和[]中是这样，在[[]]和(())是 <
  5. -ge 大于等于在test和[]中是这样，在[[]]和(())是 >=
  6. -le 小于等于在test和[]中是这样，在[[]]和(())是 <=
  • []也可以使用=, !=。注意使用<,>时使用转义字符
示例

逻辑操作符

  1. -a 相等在test和[]中是这样，在[[]]和(())是 &&
  2. -o 相等在test和[]中是这样，在[[]]和(())是 ||
  3. ! 相等在test和[]中是这样，在[[]]和(())是 !
  • []也可以使用=, !=。注意使用<,>时使用转义字符
