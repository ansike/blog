知识点总结

  • 无论是css动画，canvas动画，还是svg动画，基本思想都需要考虑一下几点：
  1. 动画当前的状态即是画布在当前时间内（浏览器渲染间隔）的一帧图像。
  2. 前提需要定时调用渲染函数不断绘制。【借助定时器或者requestAnimationFrame—最好依赖浏览器的渲染机制】
  3. 每次渲染的状态由当前动画进度决定。不管是一次性动画还是循环动画，都有一个最小环节，想清楚自己的最小环节。
  4. 每个最小环节由duration决定，受delayTime影响。

constructor

Attributes

  1. hash
  2. host
  3. hostname
  4. href
  5. origin 常用
  6. password
  7. username
  8. pathname
  9. port
  10. protocol eg:http:
  11. search

Function

  1. toString() 返回完整的URL地址，只能输出不能修改值

Static Function

  1. URL.createObjectURL(object). 将对象变成一个唯一的blob URL，参数可以是File，Blob，MediaSource对象
